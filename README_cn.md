# ESP32-Dashboard

一个基于ESP32的多功能桌面信息站，集成时钟、天气、股票、加密货币于一体，支持多种屏幕尺寸。

---

## ✨ 功能特性

- 🕐 **精准时钟** - NTP 自动同步，支持多时区显示
- 🌤️ **实时天气** - 当前天气、未来预报、气象灾害预警
- 📈 **股票追踪** - 实时股价、涨跌幅、K线趋势
- 💰 **币价监控** - 比特币、以太坊等主流加密货币价格
- 🖥️ **多屏适配** - 支持 2.8寸/4.0寸 TFT 彩屏、4.2寸三色墨水屏
- 📱 **智能配网** - 手机/网页一键配置，无需硬编码
- 🔆 **自动亮度** - 根据环境光自动调节屏幕亮度
- 🔋 **超低功耗** - 墨水屏深度睡眠模式，待机长达数月
- 🎨 **主题切换** - 多种 UI 主题，适配不同场景

## 🎨 界面展示

### 主界面
- 大号时间显示
- 日期/星期
- 当前天气图标和温度

### 天气界面
- 当前天气详情（温度、湿度、气压、风速）
- 未来 3 天预报
- 空气质量指数

### 股票界面
- 自选股列表
- 实时价格、涨跌幅
- 迷你 K 线图

### 币价界面
- 主流加密货币价格
- 24小时涨跌幅
- 市值排名
